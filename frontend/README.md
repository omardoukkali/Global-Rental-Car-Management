# Vue 3 SPA Frontend

This is the decoupled Single Page Application (SPA) frontend for the Global Rental Car Management system.

It is built entirely using:
- **Vue 3** (Composition API)
- **Vite**
- **Vue Router**
- **Pinia** (State Management)
- **Vue-i18n** (Internationalization)

## Development Setup

If you are running the project through the Docker environment, you do not need to run this manually. The Docker `web` container handles the build process automatically.

However, for local development outside Docker:

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
